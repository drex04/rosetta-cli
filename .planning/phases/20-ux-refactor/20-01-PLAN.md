# Plan 20-01 — Entry Point Unification

## Task 0: Create parent Click group

Create `rosetta/cli/__init__.py` with a `@click.group()` decorated `cli` function. Register all subcommands using **lazy imports** via `importlib.import_module` — each module is only imported when its subcommand is invoked (D-20-17). This avoids loading heavy deps like sentence-transformers and morph-kgc for unrelated commands.

Pattern:
```python
import click
import importlib

class LazyGroup(click.Group):
    def __init__(self, *args, lazy_subcommands: dict[str, str] | None = None, **kwargs):
        super().__init__(*args, **kwargs)
        self.lazy_subcommands = lazy_subcommands or {}

    def list_commands(self, ctx: click.Context) -> list[str]:
        base = super().list_commands(ctx)
        lazy = sorted(self.lazy_subcommands.keys())
        return base + lazy

    def get_command(self, ctx: click.Context, cmd_name: str) -> click.BaseCommand | None:
        if cmd_name in self.lazy_subcommands:
            module_path = self.lazy_subcommands[cmd_name]
            mod = importlib.import_module(module_path)
            return mod.cli
        return super().get_command(ctx, cmd_name)
```
