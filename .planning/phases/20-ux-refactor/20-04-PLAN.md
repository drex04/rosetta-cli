# Plan 20-04 — Pipeline Citizenship & Documentation

## Task 0: SIGPIPE handler

Add a SIGPIPE handler to the parent Click group in `rosetta/cli/__init__.py`. On SIGPIPE:
- Flush stdout
- Exit silently with code 141 (128 + 13, the Unix convention)

Implementation: `signal.signal(signal.SIGPIPE, signal.SIG_DFL)` at the top of the CLI entry point, or catch `BrokenPipeError` in a try/except wrapper around the group invocation.

The standard Python pattern:
```python
import signal
signal.signal(signal.SIGPIPE, signal.SIG_DFL)
```

**Files:** `rosetta/cli/__init__.py`
**Done:** `rosetta ingest <file> -o - | head -1` exits cleanly.

## Task 1: NO_COLOR and TTY detection

Check `os.environ.get("NO_COLOR")` at CLI startup. If set (any value), disable any ANSI formatting. Also check `sys.stdout.isatty()` — if not a TTY, suppress color.

Currently the CLI uses plain `click.echo()` with no color — so this is mostly a future-proofing guard. Add a `NO_COLOR` check in the parent group that stores state in `click.Context.obj` for any future color usage.

**Files:** `rosetta/cli/__init__.py`
**Done:** `NO_COLOR=1 rosetta --help` produces no escape codes.
