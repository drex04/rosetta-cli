# Plan 20-03 — Option Standardization

## Task 0: Add verbose/quiet to parent group

In `rosetta/cli/__init__.py`, add `-v`/`--verbose` (flag) and `-q`/`--quiet` (flag) to the parent group. Store in `click.Context.obj` dict for subcommands to read.

Create a small helper (in `rosetta/core/logging_utils.py` or inline) that subcommands call to configure stderr output level:
- Normal: warnings + errors to stderr
- Verbose: progress + info + warnings + errors to stderr  
- Quiet: errors only to stderr

Use Python `logging` module with a stderr handler. Subcommands call `click.echo(..., err=True)` for output — the helper configures a filter based on verbose/quiet state.

**Files:** `rosetta/cli/__init__.py`
**Done:** `rosetta -v ingest --help` and `rosetta -q ingest --help` work without error.
