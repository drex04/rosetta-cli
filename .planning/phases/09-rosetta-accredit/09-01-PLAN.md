# Phase 9 Plan 01 — rosetta-accredit State Machine

## Context

Phase 5 is complete (type infrastructure + Pydantic models). Phases 6–8 are planned but not yet built; phase 9 is independent of those — accredit has no dependency on RML or provenance. The existing `rosetta/cli/accredit.py` is a stub (`sys.exit(1)`). All new code must pass basedpyright strict mode.

Ledger location: `store/ledger.json`. State machine: `pending → accredited → revoked` (see `.decisions-pending.md`).

## Task 1: Add LedgerEntry and Ledger models to rosetta/core/models.py

**File:** `rosetta/core/models.py`

**Action:** Append the following after the `# --- Embeddings ---` block:

```python
# --- Accreditation ---

from datetime import datetime

class LedgerEntry(BaseModel):
    source_uri: str
    target_uri: str
    status: Literal["pending", "accredited", "revoked"]
    timestamp: datetime
    actor: str
    notes: str = ""

class Ledger(BaseModel):
    mappings: list[LedgerEntry] = []
```

**Constraints:**
- `datetime` import must be added to the top-level imports (use `from datetime import datetime`).
- `Literal` is already imported.
- `timestamp` uses `datetime` (timezone-naive ISO-8601); use `datetime.utcnow()` when creating entries.
- No default for `mappings` field — use `default_factory` style: `mappings: list[LedgerEntry] = []` (Pydantic v2 handles this correctly with `model_fields` default).

**Done when:** `from rosetta.core.models import LedgerEntry, Ledger` succeeds and `uv run basedpyright rosetta/core/models.py` exits 0.

## Task 2: Create rosetta/core/accredit.py — ledger I/O + state transitions

**File:** `rosetta/core/accredit.py` (new file)

**Functions to implement:**

```python
def load_ledger(path: Path) -> Ledger:
    """Load ledger.json; return empty Ledger if file absent."""

def save_ledger(ledger: Ledger, path: Path) -> None:
    """Write ledger to path as pretty-printed JSON."""

def find_entry(ledger: Ledger, source_uri: str, target_uri: str) -> LedgerEntry | None:
    """Return first matching entry or None."""

def submit_mapping(
    ledger: Ledger,
    source_uri: str,
    target_uri: str,
    actor: str,
    notes: str = "",
) -> LedgerEntry:
    """
    Create a pending entry. Raise ValueError if pair already exists in any state.
    Returns the new entry (does NOT save — caller saves).
    """

def approve_mapping(ledger: Ledger, source_uri: str, target_uri: str) -> LedgerEntry:
    """
    Transition pending → accredited. Raise ValueError if entry not found or wrong state.
    Mutates the entry in-place. Returns the updated entry.
    """

def revoke_mapping(ledger: Ledger, source_uri: str, target_uri: str) -> LedgerEntry:
    """
    Transition accredited → revoked. Raise ValueError if entry not found or wrong state.
    Mutates in-place. Returns updated entry.
    """
```

**Implementation notes:**
- All functions are pure w.r.t. the filesystem — they accept/return `Ledger` objects. Only `load_ledger` and `save_ledger` touch files.
- `save_ledger` uses `ledger.model_dump(mode="json")` then `json.dumps(..., indent=2, default=str)` to handle datetime serialization.
- `load_ledger` uses `Ledger.model_validate_json(path.read_text())` when file exists.
- `find_entry` searches `ledger.mappings` by `(source_uri, target_uri)` equality.
- `submit_mapping` sets `timestamp=datetime.utcnow()`.

**Type annotations:** All parameters and returns must be explicitly annotated. Import `from __future__ import annotations` at top.

**Done when:** `uv run basedpyright rosetta/core/accredit.py` exits 0 and all functions importable.

## Task 3: Implement rosetta/cli/accredit.py — Click group with 4 subcommands

**File:** `rosetta/cli/accredit.py` (replace the stub)

**Structure:**

```python
@click.group()
@click.option("--ledger", default="store/ledger.json", show_default=True, help="Path to ledger.json")
@click.option("--config", "-c", default=None, help="Path to rosetta.toml")
@click.pass_context
def cli(ctx: click.Context, ledger: str, config: str | None) -> None:
    """Manage mapping accreditation state."""
    ctx.ensure_object(dict)
    ctx.obj["ledger_path"] = Path(ledger)

@cli.command("submit")
@click.option("--source", required=True, help="Source field URI")
@click.option("--target", required=True, help="Target ontology URI")
@click.option("--actor", default="anonymous", show_default=True, help="Submitter identity")
@click.option("--notes", default="", help="Free-text notes")
@click.pass_context
def submit(ctx: click.Context, source: str, target: str, actor: str, notes: str) -> None:
    """Submit a mapping for review (pending state)."""
    ...

@cli.command("approve")
@click.option("--source", required=True)
@click.option("--target", required=True)
@click.pass_context
def approve(ctx: click.Context, source: str, target: str) -> None:
    """Approve a pending mapping."""
    ...

@cli.command("revoke")
@click.option("--source", required=True)
@click.option("--target", required=True)
@click.pass_context
def revoke(ctx: click.Context, source: str, target: str) -> None:
    """Revoke an accredited mapping."""
    ...

@cli.command("status")
@click.option("--source", default=None, help="Filter by source URI")
@click.option("--target", default=None, help="Filter by target URI")
@click.pass_context
def status(ctx: click.Context, source: str | None, target: str | None) -> None:
    """Show accreditation status. Outputs JSON array of matching entries."""
    ...
```

**Error handling:**
- Catch `ValueError` from core functions → `click.echo(f"Error: {e}", err=True); sys.exit(1)`
- On success: `submit`/`approve`/`revoke` print `{"status": "ok", "entry": {...}}` as JSON (use `entry.model_dump(mode="json")`)
- `status` prints JSON array of matching entries filtered by source/target if provided

**Entry point:** Confirm `pyproject.toml` already has `rosetta-accredit = "rosetta.cli.accredit:cli"` in `[project.scripts]`. If not, add it.

**Done when:** `uv run rosetta-accredit --help` shows the group with 4 subcommands listed; `uv run basedpyright rosetta/cli/accredit.py` exits 0.
