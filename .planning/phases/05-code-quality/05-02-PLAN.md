## Goal

Replace bare `dict` returns in user-facing output paths with Pydantic v2 models. When done, all JSON outputs (lint, suggestions, embeddings) are constructed and validated through typed models before serialisation. Misshapen output is caught at construction time, not by the consumer.

4. `rosetta-embed` output is a validated `EmbeddingReport`; each entry has `lexical: list[float]`
5. `uv run basedpyright` still exits 0 (models integrate cleanly with Plan 01 annotations)

## Tasks

### Task 1: Add pydantic to dependencies + define models

**What:** `uv add pydantic` (main dep, not dev). Write `rosetta/core/models.py` with all model classes.

**Files created/modified:**
- `pyproject.toml` (pydantic added to `[project.dependencies]`)
- `rosetta/core/models.py` (new)

**Model definitions:**
```python
from __future__ import annotations
from typing import Literal
from pydantic import BaseModel, RootModel

# --- Lint ---

class FnmlSuggestion(BaseModel):
    function_iri: str
    label: str | None = None

class LintFinding(BaseModel):
    type: str
    severity: Literal["BLOCK", "WARNING", "INFO"]
    source_uri: str
    target_uri: str
    message: str
    fnml_suggestion: FnmlSuggestion | None = None

class LintSummary(BaseModel):
    block: int
    warning: int
    info: int

class LintReport(BaseModel):
    findings: list[LintFinding]
    summary: LintSummary

# --- Suggestions ---

class Suggestion(BaseModel):
    target_uri: str
    score: float
    anomaly: bool

class SuggestionReport(RootModel[dict[str, list[Suggestion]]]):
    pass

# --- Embeddings ---

class EmbeddingVectors(BaseModel):
    lexical: list[float]

class EmbeddingReport(RootModel[dict[str, EmbeddingVectors]]):
    pass
```

**Done when:** `from rosetta.core.models import LintReport, SuggestionReport, EmbeddingReport` imports cleanly.

### Task 3: Update rosetta-lint CLI to build and serialise LintReport

**What:** Replace the dict-accumulation pattern in `lint.py` with `LintFinding` construction; build `LintReport` before JSON output.

**Files modified:**
- `rosetta/cli/lint.py`

**Steps:**
1. Add imports: `from rosetta.core.models import LintFinding, LintSummary, LintReport`
2. Replace each `findings.append({...})` with `findings.append(LintFinding(...))`
3. Build `LintSummary` from counts
4. Replace `json.dumps({"findings": findings, "summary": summary})` with:
   ```python
   report = LintReport(findings=findings, summary=LintSummary(...))
   json.dumps(report.model_dump(mode="json"))
   ```
5. Update `findings` type annotation from `list[dict]` to `list[LintFinding]`

**Done when:** `uv run basedpyright rosetta/cli/lint.py` exits 0; existing lint tests pass.

### Task 4: Update rosetta-suggest and rosetta-embed CLIs

**What:** Replace bare dict construction in suggest.py and embed.py with typed model output.

**Files modified:**
- `rosetta/cli/suggest.py`
- `rosetta/cli/embed.py`

**suggest.py steps:**
1. Import `Suggestion`, `SuggestionReport`
2. Build `list[Suggestion]` per source URI from `rank_suggestions` output
3. Construct `SuggestionReport(root={uri: suggestions, ...})`
4. Serialise: `report.model_dump(mode="json")`

**embed.py steps:**
1. Import `EmbeddingVectors`, `EmbeddingReport`
2. Wrap each embedding result: `EmbeddingVectors(lexical=vector.tolist())`
3. Construct `EmbeddingReport(root={uri: vectors, ...})`
4. Serialise: `report.model_dump(mode="json")`

**Done when:** `uv run basedpyright rosetta/cli/suggest.py rosetta/cli/embed.py` exits 0; existing suggest + embed tests pass.
