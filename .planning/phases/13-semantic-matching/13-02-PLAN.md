# Plan 13-02: Structural Feature Extraction

## Goal

Extend `rosetta-embed` to compute a 5-float structural feature vector for every schema node, stored alongside the lexical embedding in `.embed.json`. Update `rosetta-suggest` to blend lexical and structural cosine similarities using a configurable weight.

## Tasks

### Task 1 â€” `rosetta/core/features.py` + model field

**Create `rosetta/core/features.py`:**

Implement `extract_structural_features_linkml(schema: SchemaDefinition) -> dict[str, list[float]]`.

- Returns `{node_id: [f0, f1, f2, f3, f4]}` where `node_id = "{schema.name}/{node_name}"`.
- Feature vector (in order):
  - `[0]` `is_class`: 1.0 if class, 0.0 if slot
  - `[1]` `hierarchy_depth_normalized`: depth in `is_a` chain (root = 0), normalized by max depth across all nodes in the schema (0.0 if max depth is 0 or node has no ancestors)
  - `[2]` `is_required`: 1.0 if slot `required=True`, else 0.0
  - `[3]` `is_multivalued`: 1.0 if slot `multivalued=True`, else 0.0
  - `[4]` `slot_usage_count_normalized`: for slots, count of classes whose `.slots` list includes this slot name, divided by max(1, total_class_count). For classes: 0.0.
- Compute hierarchy depths by walking `is_a` chains iteratively (not recursively) to avoid stack issues on deep schemas.
- All values in [0.0, 1.0].

**Update `rosetta/core/models.py`:**

Add `structural: list[float] = []` to `EmbeddingVectors` (after `lexical`).

```python
class EmbeddingVectors(BaseModel):
    label: str = ""
    lexical: list[float]
    structural: list[float] = []
```

**Files modified:** `rosetta/core/features.py` (new), `rosetta/core/models.py`

**Done when:**

- Each vector has exactly 5 floats, all in [0.0, 1.0]
- `EmbeddingVectors` accepts `structural=[]` without error; existing embed JSON without the key still validates

### Task 3 â€” Blend in `rank_suggestions`

**Update `rosetta/core/similarity.py`:**

Add keyword-only params to `rank_suggestions`:

```python
def rank_suggestions(
    src_uris: list[str],
    A: np.ndarray,
    master_uris: list[str],
    B: np.ndarray,
    top_k: int = 5,
    min_score: float = 0.0,
    *,
    A_struct: np.ndarray | None = None,
    B_struct: np.ndarray | None = None,
    structural_weight: float = 0.2,
) -> dict[str, Any]:
```

Blend logic:
- If both `A_struct` and `B_struct` are provided and neither is all-zero: compute `struct_sim = cosine_matrix(A_struct, B_struct)`, then `final = (1 - structural_weight) * lex_sim + structural_weight * struct_sim`.
- Otherwise: `final = lex_sim` (lexical-only fallback â€” no log, no error).
- Use `final` for sorting and score values. Round to 6 decimal places as before.

All-zero check: `np.any(A_struct != 0) and np.any(B_struct != 0)`.

**Files modified:** `rosetta/core/similarity.py`

**Done when:**
- With non-zero structural arrays, output scores differ from lexical-only scores
- With zero/None structural arrays, output is identical to lexical-only baseline
- `structural_weight=0.0` â†’ pure lexical; `structural_weight=1.0` â†’ pure structural
- `basedpyright` passes

### Task 4 â€” Wire suggest CLI + `rosetta.toml`

**Update `rosetta/cli/suggest.py`:**

Extract structural arrays from the parsed `EmbeddingReport` and pass to `rank_suggestions`:

```python
resolved_structural_weight = float(
    get_config_value(cfg, "suggest", "structural_weight", cli_value=None) or 0.2
)

# Build structural numpy arrays (empty structural â†’ zero-row matrix â†’ fallback triggers)
src_structs = [src_report.root[u].structural for u in src_uris]
master_structs = [master_report.root[u].structural for u in master_uris]

struct_dim = max((len(v) for v in src_structs + master_structs), default=0)
if struct_dim > 0:
    A_struct: np.ndarray | None = np.array(
        [v if v else [0.0] * struct_dim for v in src_structs], dtype=np.float32
    )
    B_struct: np.ndarray | None = np.array(
        [v if v else [0.0] * struct_dim for v in master_structs], dtype=np.float32
    )
else:
    A_struct = None
    B_struct = None

result = rank_suggestions(
    src_uris, A, master_uris, B,
    resolved_top_k, resolved_min_score,
    A_struct=A_struct, B_struct=B_struct,
    structural_weight=resolved_structural_weight,
)
```

**Update `rosetta.toml`:**

Add to `[suggest]`:


**Files modified:** `rosetta/cli/suggest.py`, `rosetta.toml`
