# Phase 21 Plan 01: HumanCuration Filtering

## Task 1: Add `filter_decided_suggestions()` + delete derank functions

**Files:** `rosetta/core/similarity.py`

1. Delete `import copy` (L3) — only used by `apply_sssom_feedback`
2. Delete `_adjusted_score()` (L102-114)
3. Delete `apply_sssom_feedback()` (L117-149)
4. Add `filter_decided_suggestions(result, log)`:
   - Signature: `def filter_decided_suggestions(result: dict[str, Any], log: list[SSSOMRow]) -> dict[str, Any]`
   - Returns a **new dict** (does not mutate input) — consistent with the old `apply_sssom_feedback` copy pattern
   - Build `hc_approved_subjects: set[str]` — subjects with HC + predicate != `owl:differentFrom`
   - Build `hc_rejected_pairs: set[tuple[str, str]]` — (subject, object) pairs with HC + `owl:differentFrom`
   - For each src_uri in result: if src_uri in `hc_approved_subjects`, skip entirely; otherwise copy entry with suggestions filtered to exclude `hc_rejected_pairs`
   - Empty log → both sets empty → return full result unchanged (but still a new dict)

**Done when:** `filter_decided_suggestions` exists; `apply_sssom_feedback`, `_adjusted_score`, and `import copy` do not; `uv run basedpyright` clean on the file.

2. Add auto-create logic with error handling:
   ```python

   if not lp.exists():
   ```
